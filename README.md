# 📈 Sales Forecasting Using Prophet

## 📌 Project Overview

This project predicts future sales using historical Walmart sales data with the Facebook Prophet time-series forecasting model. It analyzes historical sales patterns and generates future sales forecasts by considering trends and seasonality.

---

## 📂 Dataset

The project uses the Walmart Sales Forecast dataset.

Files:
- train.csv
- features.csv
- stores.csv

---

## 🚀 Technologies Used

- Python
- Pandas
- Prophet (Facebook Prophet)
- Matplotlib
- Scikit-learn

---

## ✨ Features

- Data Cleaning
- Data Merging
- Time Series Forecasting
- Trend Analysis
- Yearly Seasonality Analysis
- Forecast vs Actual Sales Comparison
- Sales Forecast Visualization
- Model Evaluation using MAE and RMSE

---

## 📊 Output

The project generates:

- Sales Forecast Graph
- Actual vs Forecast Sales Graph
- Trend Analysis
- Yearly Seasonality Graph

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/kasaudhanakash/Sales-Forecasting-Using-Prophet.git
```

Install required libraries

```bash
pip install pandas matplotlib prophet scikit-learn
```

Run the project

```bash
python main.py
```
